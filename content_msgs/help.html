Egon Text Editor - Help

Marking

Egon Text Editor works quite differently from other text editing programs, something that can work differently between each character you will choose to change. The simplest way to change something text-related is by marking it.

If you do not mark something, Egon Text Editor will select automatically for you what it will affect depending on what tool you're willing to use:

  • Bold, Italics, Underline, text color, text to speech, and text twisting function - selects the entire file.
  • Align right, Align center, Align left - selects the current line.

Import non-text file

Egon-Text-Editor lets you import some nontext file types to your text file. Press the "import file" button in the file menu and select a supported file (Excel, CSV, JSON, XML). The file is converted to text and appended at the current cursor position.

Local vs. Global file

  • Local: a file on your computer, accessed via a menu.
  • Global: a file available on the internet, accessed via a link.

File bar & Status bar

The File bar and status bar are two labels at the bottom of the program. They notify you about:

  • Status bar: number of words and characters.
  • File bar: file name, date and time of the last save, file state, etc.

You can turn off these labels in Settings.

Search online

Search Online opens a new tab in your default browser and searches by domain.

The "copy from file" button has two behaviors:

  1. If text is marked, only the marked text is copied to the text box.
  2. If text is not marked, the whole text file is copied to the text box.

Custom U.I. colors

There are three options to change predefined colors to your custom ones (aside from text color):

  1. Button colors - buttons' background color.
  2. Menu colors - menus' background and text colors.
  3. App colors - text box color, status & file bar colors, toolbar colors, frame color, text color (until version 1.0.9).

Emoji detection

The emoji detection feature is automatic by default. You can also detect emojis manually via the "detect emojis" button in the settings menu.

Sort by words/characters

These functions sort values that aren't numbers by using their ASCII values.

N.L.P. menu

The N.L.P. menu is attributed to the Natural Language Processor: it processes the main text box widget's text and finds something based on the command you pressed.

Top most

The Top most setting is in the options menu. When activated, it makes the main window's Z-order above any other window.

Connection with OpenAI products

In the program, you can use OpenAI products, which include ChatGPT and DallE. You connect via a login window that requests your OpenAI / Azure-OpenAI keys. After this, a special window opens for the specific A.I. you chose.

Note: API usage may incur costs charged by the provider.

Text decorators

Text decorators are styles of large text using regular text. Available styles: binary, bash, and astrix.

  • Outputs can be vertical or horizontal.
  • Convert text from the main text box or use the tool's own text box.
  • Paste results directly to the main text box.

Text twisting functions

Text twisting functions change the order of parts of the text in straightforward ways (for fun or exploration). Located at the bottom of the Edit menu.

  • Reverse characters
  • Reverse words
  • Join words
  • Upper / Lower
  • Sort by characters / Sort by words

Dev mode

Dev mode can be activated via Options. When activated, it lets you access the Record function.

Record

The Record function keeps track of main events that occur in the app. When you open the tool, events are written in an organized manner with time and details on different lines.

Special tools option

The "special tool" option (enabled by default) appears in Options. If you turn it off, you will no longer have access to the Tools and Colors+ menus.

Tools' font

Some tools' font is bold while others are not. Bold indicates tools that are the finest and most original.

IDE

If the program detects that you are using a Python file, it will add several options:

  1. Run the code directly from the program.
  2. The main text box will share space with a console.
  3. Additional options to ease usage.

Saved settings

Most program options are saved in a file in your E.T.E. directory. Reset all options via Advanced options -> Restore defaults.

If the saved settings file prevents the program from opening/working, a special window will ask for permission to delete it and create a new default file.

Night modes

There are two night modes: one with mainly blue colors and one with mainly black colors (the original "Dracula" mode, which is the default).

Affected windows include:

  • Main EgonTE window
  • Advanced options window
  • Help & Patch notes "information" window

Opening sentence

If the program doesn't automatically open your last saved file, it will write a sentence for you at startup. The sentence can be one written for you or one from an API showcasing quotes by special individuals.

You can also get a sentence directly in the main text box via Tools -> "Inspirational quote".

Random name

The program contains a random name function. On first opening the tool, you can generate names immediately, copy the name, or re-roll.

Advanced re-rolling

  • Choose the gender.
  • Choose the part of the name you want to get.

Dictionary/Wikipedia

Expand your knowledge using the Dictionary/Wikipedia tools (Tools menu). Search with a term and see results in the tool's text box, with a button to paste directly to the main text box.

Wikipedia tool only: includes filtering options based on which part of the page you want.

Generate sequence

Accessed via the Tools menu. In a small control window, you can determine the number of characters and whether to include/exclude symbols. Use the Generate button to create characters.

Correct writing

Accessed via the Edit menu. This function takes your text and replaces it with a corrected version using a corrector from a library.

Important: Not recommended when your text is already correct, as it may change words you intended.

Calculator tool bonus U.I.

The calculator tool has extra U.I.: number buttons and clear buttons appear on the left when you press "Calculator U.I.". In the "show expression" frame, press them to add to the entry.

YouTube transcript

The program has transcript functions: YouTube transcript and W.A.V. file transcript (file transcript is W.I.P.).

For YouTube transcript, provide the video's ID (the part at the end of the link). The program opens a text box showing what is spoken, start times, and iteration of items.

Library installer

This tool opens when not all required libraries are installed. From there, install required and optional libraries, and upgrade pip.

When the process completes, it will show how many libraries failed to install. The Install button is then replaced with a Restart button.

Search tools

Search tools include "Find text" and the U.I. at the bottom of the "knowledge window".

  • Automatic search box shows matching terms count and provides navigation: down, reset, tag/untag all occurrences.
  • "Find text" also includes an interactive list of the most used terms.

Shortcuts

Main window

  • Ctrl + O - open file
  • Ctrl + S - save file
  • Ctrl + N - new file
  • Alt + F4 - close the app
  • Ctrl + A - select all text
  • Ctrl + B - bold text
  • Ctrl + I - italics text
  • Ctrl + U - underline text
  • Ctrl + L - align text left
  • Ctrl + E - align text center
  • Ctrl + R - align text right
  • Ctrl + P - print file
  • Ctrl + Delete - clear text
  • Ctrl + F - find text tool
  • Ctrl + H - replace text
  • Ctrl + G - go to (text item)
  • Ctrl + Shift + J - join words
  • Ctrl + Shift + U - lower / upper text
  • Ctrl + Shift + R - reverse characters
  • Ctrl + Shift + C - reverse words
  • Alt + D - copy file path name
  • Ctrl + + - font size up
  • Ctrl + - - font size down
  • F5 - output current date/time
  • F11 - fullscreen mode

Tools

Drawing tool

  • Arrow keys - move the drawing around
  • Ctrl + . - pen size up (alt via settings: mouse wheel up)
  • Ctrl + , - pen size down (alt via settings: mouse wheel down)

Wikipedia tool (image mode)

  • Ctrl + . - move image forwards
  • Ctrl + , - move image backwards
