[ 1.13.2 - Tools revolution 2 - 23/8/25 (Summary) ]

OpenAI interfaces (ChatGPT and Dall-E)

  • Unified into a single standalone module: ai_popups.py
  • No UI freezes, safer threading, and proper streaming
  • Cleaner, more discoverable controls; faster iteration with bypass mode
  • Consistent with enhanced popup builders and modernized across both tools

Natural language processor

  • Same feature set, faster and more reliable
  • Single entry function to open the tool; fewer moving parts
  • Better error handling with time and size safety
  • Consistent UI using standard popup builders; improved naming for maintainability

Email tool

  • Modernized: safer connection, better validation
  • Supports CC, BCC, attachments, and body statistics
  • Saved fields, login test, and friendlier UI
  • Standalone file with a one-line forwarder for easy access

Auto-lists

  • No thread and no full-buffer scan; logic is local to the current line at the caret
  • Reliable Tk indices and correct event signature (handler receives event and returns "break" when handled)
  • Flexible marker support with clear rules; simple to extend
  • Clean on/off via self.aul without duplicating bindings
