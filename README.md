## Features 🔥
Work with files - save, open, make a new & many more 💾.  
Advanced file actions - print, compare, statistics, import, merge  🖨️.  
Change the style of your text with fonts, sizes, colors & typefaces 🗚.  
Listen to your writings & write through speaking 📣.  
Edit your text with many ordinary, advanced & fun tools 🔧.  
Stunning extras - web scraper, calculator & many more 📩.  
Analyze your document with a natural language processor 📄.  
Program with an integrated development environment 🤖.  
Many settings and customization options to the software itself ⚙️.  
Adjustments to the users - unique modes, saved options, self-repair 🖇️.  
Enjoy two unique night modes 🌙.  

## Pop-up Toolkit 🧰
EgonTE includes over 20 mini-apps to help with every task:

| Category | Tools |
| :--- | :--- |
| **Utilities** | Calculator, Virtual Keyboard, Clipboard History, Unit Converter |
| **Web** | Web Scraper, Weather App, Google Translate, Email Client |
| **Dev** | Git Manager, Encryption, Regex Helper, Template Generator |
| **Creative** | Handwriting Pad, Text Decorators, Symbol Picker |

## How to Run 🏃
1. Install Python 3.10+
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python EgonTE.py`
